You Spin Me Round

by Yazhini Priyadharshini Rajesh

In this project we are going to take a look at data about the relative strength and damage from tornadoes across the state and across the last 65 years. Taken From this website.

About the data The data is about 7MB total for the entire US for the last 65 years covering about 62,000 tornadoes for the entire US over that period, but only about 2,500 in IL The data we are going to focus on includes: date time state magnitude injuries fatalities loss start lat + lon end lat + lon length width FIPS county codes.

The Data

This application visualizes the following:

  • About the Project: The user can choose any state from a list and the year from the list
  • Tornadoes- Yearly This tab shows how the tornadoes were situated overall on a yearly basis. It includes a pie chart which shows the percentage as a basis for comparison
  • Tornadoes- Monthly This tab shows how the tornadoes were situated overall on the basis for months. This tab can be used to predict where there is a particular season triggers the presence of tornadoes
  • Tornadoes- StateWise Now that we have seen the yearly and monthly overview of tornadoes we next look at the state wise trends of tornadoes
  • Tornadoes- CountyWise We then look at the countywise overview. By selecting the state from the sidebar panel we can view the bar graph and a table of the overview for the counties.
  • Map- State Wise State Wise map is a Choloropleth map of the selected state which shows the death, loss, magnitude and injuries of the state on the map of the state
  • Map- Tornadoes Last but not the least, we can view a map of the USA which shows the intensity and the locations of tornadoes in the US on a yearly basis. We can choose our preference of magnitude and what parameter is in question

Application

This application is written in R, using the shiny dashboard framework. Graphs use ggplot2 and plotly; the map view uses leaflet and location data from addTiles. In addition, tidyr and jpeg libraries are used to support corresponding functions.

It is designed to be viewed as a dashboard with no scrolling. Select the items on the left navigation pane to switch between various views. We can also minimize the sidepad by using the minimize button on top next to the title.

Items in the legend may be clicked to show/hide the item from the graph. Double-click on a legend item to show only that item. Hover over chart elements for more information about the point, bar or pie.

I have included the presence of functions and reactives to make the code more interactive and for faster processing.
